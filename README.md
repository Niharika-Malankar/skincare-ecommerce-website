
# ✨ Lumora – Skincare E-Commerce Website

A modern and elegant skincare e-commerce website built with **React** and **Vite**, designed to provide a premium beauty-shopping experience through clean layouts, luxury-inspired typography, and visually appealing product showcases.

## 📌 Project Overview

Lumora is a front-end skincare e-commerce website that focuses on aesthetic design, responsive layouts, and user-friendly navigation. The website features a luxury skincare theme with carefully selected color palettes, custom product imagery, and elegant typography using Google Fonts.

## 🚀 Features

* Responsive Navigation Bar
* Luxury Hero Section with Featured Model Image
* Product Collection Showcase
* Interactive Product Cards
* Custom Icon-Based Feature Highlights
* Newsletter Subscription Section
* Modern Footer Design
* Premium Skincare Branding
* Hover Effects and Smooth UI Interactions

## 🛠 Technologies Used

* React.js
* JavaScript (ES6+)
* Vite
* CSS Inline Styling
* Google Fonts (Cormorant Garamond)
* HTML5
