# Skincare E-Commerce Website

A modern skincare e-commerce website built using React and Vite.

## Features
- Responsive navigation bar
- Hero section
- Product showcase
- Shopping cart interface
- Newsletter subscription section

## Technologies Used
- React
- JavaScript
- Vite
- CSS (Inline Styling)
